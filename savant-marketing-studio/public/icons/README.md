# PWA Icon Requirements

Replace these placeholder icons with branded versions of your logo.

## Required Icons

| File | Size | Purpose |
|------|------|---------|
| `icon-16.png` | 16x16 | Browser favicon (small) |
| `icon-32.png` | 32x32 | Browser favicon (large) |
| `icon-192.png` | 192x192 | Android home screen icon |
| `icon-512.png` | 512x512 | Android splash screen / PWA install |
| `../apple-touch-icon.png` | 180x180 | iOS home screen icon |

## Design Guidelines

- Use PNG format with transparency
- Icons should be square
- For maskable icons (Android), keep important content within the safe zone (center 80%)
- iOS icons should NOT have transparency (iOS adds rounded corners automatically)
